# 地図マーカーアプリ

CSVファイルから地図上にマーカーを表示するWebアプリケーションです。OpenStreetMapを使用し、マーカーの位置情報を視覚的に確認できます。

## 機能

- OpenStreetMapを使用した地図表示
- CSVファイルからのマーカー読み込み
- マーカーのクラスタリング表示
- マーカークリック時のポップアップ表示
- 全画面表示モード
- サンプルCSVのダウンロード
- レスポンシブデザイン対応

## 使用方法

1. アプリケーションを開く
2. 「サンプルCSVをダウンロード」ボタンをクリックしてサンプルファイルを取得
3. CSVファイルをアップロードすると、地図上にマーカーが表示されます
4. マーカーをクリックすると、ラベルと説明文が表示されます
5. 地図の左上の全画面表示ボタンで全画面モードに切り替え可能

## CSVファイル形式

以下の形式のCSVファイルを使用してください：

```csv
latitude,longitude,label,description
35.6812,139.7671,東京駅,東京の中心駅
35.6895,139.6917,新宿駅,新宿の中心駅
35.6586,139.7454,渋谷駅,渋谷の中心駅
```

- `latitude`: 緯度（必須）
- `longitude`: 経度（必須）
- `label`: マーカーのラベル（必須）
- `description`: マーカーの説明文（必須）

## 開発環境のセットアップ

1. リポジトリをクローン
```bash
git clone [リポジトリURL]
```

2. ローカルサーバーを起動
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx serve
```

3. ブラウザで `http://localhost:8000` にアクセス

## 注意事項

- 大量のマーカーを表示する場合は、クラスタリング機能により自動的にグループ化されます
- 地図の表示にはインターネット接続が必要です
- CSVファイルはUTF-8エンコーディングで保存してください
